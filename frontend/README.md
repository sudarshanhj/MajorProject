# DeepStegAI Cinematic Intelligence Suite

This is the modern React frontend for DeepStegAI, designed for elite cybersecurity intelligence operations.

## Tech Stack
- **React 18** (Vite)
- **TypeScript**
- **TailwindCSS** (Obsidian Industrial Theme)
- **Framer Motion** (Motion Design)
- **Three.js / React Three Fiber** (3D Visuals)
- **Zustand** (Global State)
- **Axios** (API Layer)

## Interface Overview
- **Landing Page**: Generative Neural Point Cloud Sphere.
- **AI Detection**: 3D Volumetric Scan System.
- **Embed/Extract**: Forensic-grade data hiding and recovery.
- **Batch Operations**: Scalable intelligence workflows.
- **Admin Shell**: Secure command console.

## Getting Started
1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
3. Ensure the Flask API is running on `http://127.0.0.1:5000` (configured via Vite proxy).

## Production Build
1. Run `npm run build`
2. Serve the `dist` folder.
